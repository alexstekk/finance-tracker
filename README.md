## Finance Tracker with Next.js

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Stack

- Next.js
- Auth with Clerk
- TypeScript
- Drizzle ORM
- Tailwind
- PostgreSQL
